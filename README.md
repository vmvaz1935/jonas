# Cartilha de Fisioterapia - Jonas Pulcheri

Cartilha de exercícios de fisioterapia para pós-operatório de joelho.

## Acesso

O acesso aos exercícios é protegido por senha.

## Configuração do GitHub Pages

Para ativar o GitHub Pages automaticamente:

1. Acesse o repositório no GitHub: https://github.com/vmvaz1935/jonas
2. Vá em **Settings** > **Pages**
3. Em **Source**, selecione **Deploy from a branch**
4. Escolha a branch **main** e a pasta **/ (root)**
5. Clique em **Save**

O site estará disponível em: `https://vmvaz1935.github.io/jonas/`

## Arquivos

- `index.html` - Página principal com os exercícios (protegida por senha)
- `relatorio.html` - Relatório de avaliação do paciente
